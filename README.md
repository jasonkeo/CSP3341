# CSP3341

## App Summary
The application is a Java-based desktop system designed to manage and retrieve student access records. It provides a graphical user interface (GUI) built using the Swing library. The system also includes a login feature supporting both user and admin roles. Its main purpose is to allow efficient interaction with stored student data through a simple and lightweight interface.

## Functional Requirements
The system is designed to meet the following functional requirements:

- The system must allow users to search for student access records based on specific criteria  
- The system must enable admins to insert new student records into the database  
- The system must provide a graphical user interface for interacting with stored data  
- The system must include a secure login system  


## How to Use

### Prerequisites
- Java (Version 26 or compatible)  
- Maven installed and configured  

### 1. Installation
1.1 Clone or download the repository  
1.2 Navigate to the project directory  

### 2. Build
Run the following command to compile and package the application: 'mvn clean package'

### 3. Run
After building, run the generated `.jar` file: 'java -jar target/<your-jar-file>.jar'
